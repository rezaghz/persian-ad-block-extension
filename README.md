# Persian Ad Block extension

![](assets/bnr.png)

Persian Ad Block یه افزونه است که میاد تمام تبلیغات های مزخرف شرکت های تبلیغاتی پاک میکنه و دیگه شما هیچ تبلیغاتی رو توی صفحه نمی بینین

## نصب اکستنشن
